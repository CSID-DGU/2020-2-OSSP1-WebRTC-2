# 동국대학교 공개 SW 프로젝트 
> 2020-2-OSSP1-WebRTC-2
> WebRTC를 이용한 화상 회의 시스템 구현 프로젝트

 ## 사용법
 ### master branch 기준
  ```
  git clone https://github.com/CSID-DGU/2020-2-OSSP1-WebRTC-2.git
  cd conference
  npm start
 ```
 ### 혹은 웹 페이지에서 
 ### http://localhost:3001 입력하여 실행
